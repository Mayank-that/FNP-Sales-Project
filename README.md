# 📊 FNP Sales Analysis Dashboard (Excel Project)

## 📌 Overview
This project focuses on analyzing sales data from FNP (Ferns N Petals) to uncover meaningful business insights and trends. The dashboard was built entirely in Microsoft Excel using data cleaning, Pivot Tables, Pivot Charts, Slicers, and analytical techniques.

The objective was to transform raw sales data into an interactive dashboard that enables better decision-making through sales performance analysis.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Identify top-performing products and categories.
- Track monthly revenue trends.
- Understand occasion-based purchasing behavior.
- Evaluate city-wise order distribution.
- Measure delivery performance.
- Explore the relationship between order quantity and delivery time.

---

## 📈 Key Metrics

- Total Revenue
- Total Orders
- Average Revenue per Order
- Average Delivery Time
- Correlation Between Order Quantity and Delivery Time

---

## 🔍 Key Insights

- August generated the highest revenue among all months.
- The Colors category contributed the largest share of revenue.
- Anniversary and Raksha Bandhan occasions drove significant sales.
- Order quantity and delivery time showed almost no relationship (Correlation ≈ 0.003).
- A small number of cities contributed a large share of total orders.
